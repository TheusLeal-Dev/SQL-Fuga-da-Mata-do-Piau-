Capítulo 1 — Fome na mata

HISTÓRIA

Você acorda no meio da mata, fraco e com fome. Depois de caminhar entre as árvores, encontra um cesto com frutas e um terminal antigo coberto de folhas.

O terminal parece ter registros em uma tabela chamada frutas_encontradas. Nem tudo é seguro. Você precisa sobreviver.

Saída do Terminal

Dica: tente executar um SELECT da tabela frutas_encontradas.
Esquema: frutas_encontradas (id, nome, origem, comestivel, alerta)

Nenhum resultado ainda. Execute um SELECT no terminal pra ver os dados.

TERMINAL SQL

MISSÃO: Descubra quais frutas você encontrou. Comece listando o conteúdo da tabela.

Exemplos: SELECT * FROM frutas_encontradas; SELECT nome, origem FROM frutas_encontradas WHERE comestivel = 1;

Log Aguardando comando... 
Digite um SELECT para continuar.
SUCESSO

Você sobreviveu.

Você separou as frutas seguras e recuperou energia. No chão, um mapa rasgado aparece… mas as coordenadas estão erradas.
